TM Machine¶

The TM Machine is a block which allows creating TMs for moves which your pokémob already knows, or should have known at a lower level. This acts as a way to both copy TMs, as well as a move relearner.

To use the TM Machine, you need a TM, and a filled pokécube.

When you first open the GUI, there are no displayed moves. To add moves to the list, you can place a filled pokécube into the pokécube slot of the machine.

The < and > buttons can be used to cycle through the list is available moves, and then the apply button will map the selected move onto the TM in the TM slot. The TM used does not need to be blank, and this can be used to overwrite un-wanted TMs.
